# Inteligência Artificial

Este repositório contém os exercícios e projetos desenvolvidos na disciplina de Inteligência Artificial, com foco no estudo de técnicas e aplicações de IA na resolução de problemas.

O objetivo é praticar e consolidar conceitos como representação do conhecimento, aprendizagem de máquina, algoritmos inteligentes e construção de sistemas capazes de tomar decisões.

💡 Os projetos envolvem tanto conceitos teóricos quanto aplicações práticas utilizando diferentes abordagens de inteligência artificial.

---

## 📚 Conteúdo
- Conceitos básicos de Inteligência Artificial  
- Representação do conhecimento  
- Estados e busca heurística  
- Lógica de primeira ordem  
- Resolução de problemas  
- Sistemas especialistas  
- Algoritmos genéticos  
- Aprendizagem automática (machine learning)  
- Redes neurais artificiais  
- Agentes inteligentes e sistemas multiagentes
